# â–¶ï¸ Run & Build the Next.js App

After setup, navigate into your project folder and run the following commands:

Start Development Server:
npm run dev
(Open http://localhost:3000 in your browser)

Create Production Build:
npm run build
(Builds the app for production. Output will be in the .next/ folder.)

Start Production Server:
npm start
(Starts the production build locally on http://localhost:3000)

Lint Your Code (if ESLint was enabled):
npm run lint
(Runs ESLint to find and fix code issues.)
